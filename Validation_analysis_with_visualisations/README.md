# Validation analysis with visualisations

## Contains

- analysis_model_agreement.py: model agreement and vote summaries.
- analysis_year_trends.py: yearly trend plots.
- analysis_explicit_ukb_mentions.py: explicit UKB mention checks.
- analysis_keyword_categories.py: keyword/category summaries.
- analysis_tfidf_terms.py: TF-IDF discriminative terms.
- analysis_semantic_map.py: semantic map and SI/silhouette index.
- run_all_visualisations.py: runs all visualisation modules.
- common.py: shared helpers.

## Run

pip install -r requirements.txt

python run_all_visualisations.py

Edit placeholder input/output paths before running.
